# AI模型 7.1.1 项目结构说明

当前版本：7.1.1 盘口订单簿系统接入

当前阶段在信号页固定接入 Binance 公共盘口订单簿，为后续市场结构、机会榜、AI交易委员会和自动交易系统提供基础数据。

## 文件结构

```text
AI模型_7.1.1/
├─ app.py
├─ requirements.txt
├─ services/
│  ├─ __init__.py
│  ├─ binance_public.py
│  ├─ kline_service.py
│  ├─ orderbook_service.py
│  ├─ orderbook_analyzer.py
│  ├─ market_cache.py
│  └─ background_refresher.py
├─ 项目结构说明.md
└─ 运行说明.md
```

## 当前版本已完成

- 信号页固定布局：K线图 → 盘口订单簿 → 市场结构 / AI分析 / 交易信号预留。
- 新增 `services/orderbook_service.py` 获取 Binance 公共盘口深度。
- 新增 `services/orderbook_analyzer.py` 统一输出盘口分析字段。
- `market_cache.py` 新增盘口缓存、盘口状态、盘口更新时间和盘口错误信息。
- `background_refresher.py` 新增当前交易对象盘口 1 秒后台刷新。
- 盘口默认显示买卖各5档。
- 展开后显示买卖各10档。
- 显示价格、数量、累计数量。
- 显示深度条，挂单越大条形越长。
- 显示买盘比例、卖盘比例和多空倾向。
- 显示大买单、大卖单与距离当前价格。
- 切换交易对象后，盘口跟随全局当前交易对象刷新。

## 当前版本禁止内容

- 不接入 DeepSeek / Gemini / GPT
- 不接入模拟交易
- 不接入实盘交易
- 不接入账户 API
- 不接入 WebSocket

## 后续预留

`orderbook_analyzer.py` 输出的买盘强度、卖盘强度、买卖比、支撑位、压力位、大单监控和多空倾向，可供后续市场结构、机会榜和AI交易委员会直接调用。
